# 견적 및 범위 제안

## 결론

이 의뢰는 단순 웹페이지 1개 제작이 아니라, Google Drive, Make.com, OpenAI API, Airtable, Softr, SpeakMate 앱을 연결하는 업무 자동화 구축이다.

MVP 기준으로는 Make.com 운영이 가능하다. 다만 완전 무관리 자동화는 아니며, 월 1회 로그 확인 정도의 운영은 필요하다.

## 권장 견적

### 옵션 A. 노코드 MVP

범위:

- Airtable DB 설계
- Google Drive 피드백 업로드 자동 감지
- OpenAI 피드백 분석
- Airtable 저장
- Softr 학생/원장 기본 대시보드
- 운영 가이드

예상 공수:

```text
25~40시간
```

권장 견적:

```text
150만~300만 원
```

### 옵션 B. SpeakMate 연동 포함

범위:

- 옵션 A 전체
- SpeakMate Academy webhook 저장 연동
- 말하기 연습 결과 Airtable 저장
- 녹음 파일 Google Drive 저장
- 학생 활동 로그
- 관리자 통계 기초 화면

예상 공수:

```text
45~70시간
```

권장 견적:

```text
300만~500만 원
```

### 옵션 C. 운영 안정화 포함

범위:

- 옵션 B 전체
- 2주 안정화
- 실패 케이스 수정
- 강사 문서 템플릿 보정
- 원장 교육 1회

권장 견적:

```text
500만~700만 원
```

## 월 유지보수

유지보수를 맡지 않고 끝내는 것도 가능하다. 다만 아래 조건이 필요하다.

- 운영 가이드 제공
- 계정/API key를 원장 명의로 이관
- Make 실패 로그 확인 방법 교육
- Airtable 필드명 변경 금지 안내

유지보수를 맡는다면 가벼운 점검형이 적당하다.

```text
월 10만~30만 원: 월 1회 로그 점검, 간단한 오류 재실행
월 30만~60만 원: 월 1회 리포트, 작은 필드/화면 수정 포함
건별 지원: 시간당 5만~10만 원
```

## SaaS 비용은 별도

견적에는 SaaS 구독료와 OpenAI API 사용료를 포함하지 않는 것이 좋다.

2026-06-14 기준 공식 가격표를 확인해 제안서에 별도 표기한다.

- Make.com: Free는 1,000 credits/month, Core는 10,000 credits/month 기준 월 $9부터
- Airtable: Free 가능, Team은 연간 결제 기준 $20/user/month, Business는 $45/user/month
- Softr: 학생 로그인 대시보드를 제대로 운영하려면 유료 플랜 검토 필요
- OpenAI API: 문서 길이와 사용 모델에 따라 종량 과금

## 제외 범위로 명시할 것

- 학생 개인정보보호 법률 검토
- 고급 통계/성적 분석 리포트
- 완전한 LMS 개발
- 네이티브 모바일 앱
- 24시간 장애 대응
- OpenAI 결과의 교육적 최종 검수
- SaaS 서비스 자체 장애 대응
